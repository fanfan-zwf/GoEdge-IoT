# GoEdge IoT

**基于 Go 语言的分布式工业物联网监控平台**

# 一句话定位

基于 Go 语言开发的**分布式、云端一体化工业物联网平台**，采用 **InfluxDB 高性能时序数据库**，实现 PLC 数据采集、视频监控、实时展示、历史数据查询、边缘计算与精细化权限管理，完美支持 4G / 跨网、多现场、大规模设备接入。

# 核心功能

1. **PLC 远程数据采集**

   支持主流 PLC 协议，跨局域网、4G 无线稳定采集，边缘端独立运行，断网续传。

   

2. **云端统一点位配置管理**

   所有点位、采集频率、报警规则**全部在云端配置**，边缘自动同步，无需现场操作。

   

3. **边缘计算 + 本地报警**

   边缘端实时计算报警，不依赖网络，保障工业场景安全可靠。

   

4. **实时数据推送 + 浏览器展示**

   基于 WebSocket 实时推送数据，低延迟、高并发，前端秒级刷新。

   

5. **InfluxDB 时序数据存储（历史数据核心）**

   采用 **InfluxDB 专业时序数据库**，高效存储海量时间序列数据，支持：

   

   - 超高写入速度

   - 秒级历史数据查询

   - 按时间范围聚合、统计、降采样

   - 历史曲线、趋势分析、数据追溯

     

     特别适合温度、压力、流量、电流等工业数据长期存储。

   

6. **海康 / 大华视频实时预览**

   对接官方 API，RTSP 转 Web 码流，浏览器直接播放视频监控。

   

7. **完善权限体系**

   用户、用户组、组长三级管理，支持**点位级读写权限控制**，安全可控。

   

8. **权限扩展功能**

   `可根据权限主题自由开发其他所需功能`

   

9. **全分布式微服务架构**

   服务拆分清晰，可独立部署、按需扩容：

   

   - 点位配置服务
   - 边缘采集服务
   - 实时数据服务
   - MQTT 通信服务
   - 报警推送服务
   - 用户权限服务

   

# 核心技术亮点

- **纯 Go 开发**：性能强、部署简单、跨平台、资源占用极低。
- **InfluxDB 时序存储**：工业级历史数据方案，海量数据不卡顿。
- **MQTT 数据总线**：适配 4G / 弱网 / 跨网，稳定、轻量、可靠。
- **云端配置 + 边缘执行**：配置统一管理，边缘只负责采集。
- **分布式微服务**：服务解耦，高可用、易扩展、易维护。
- **视频 + 数据 + 报警一体化**：一套平台搞定工业现场全部需求。

# 适用场景

- 工厂设备远程监控
- 多站点、跨地域集中管理
- 4G / 无线通信的工业场景
- 需要长期存储历史数据、做数据分析的项目
- 高权限要求、多用户分级管理的物联网平台

# 平台价值

- 运维成本低：云端统一配置，无需现场调试
- 数据安全可靠：InfluxDB 保证历史数据不丢失
- 系统稳定：边缘断网仍可正常采集、报警
- 扩展性强：支持不断接入新设备、新功能
- 体验流畅：实时数据 + 历史曲线 + 视频同屏展示
